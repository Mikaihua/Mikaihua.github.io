1、使用命令创建名称为blog文件夹
2、cd到blog文件夹中
3、按shift+右键，打开命令输入：npm init
a、name项目名称 
b、bersion版本号
c、description项目说明描述
d、entry point项目入口
e、author作者

不填就是默认值，填就是自定值

4、安装express：npm/cnpm install express

5、express -v

6、在blog目录下新建index.js

7、在index.js中：
var express = new require("express");

var app = express();

app.get("/",function(req,res){
	res.send("Hello World!");
});

app.listen(3000);

8、在命令行中输入node index 启动服务器

9、在浏览器中打开localhost:3000查看
