冒泡流与捕获流

事件捕获阶段：事件从最上一级标签开始往下查找，直到捕获到事件目标(target)。
事件冒泡阶段：事件从事件目标(target)开始，往上冒泡直到页面的最上一级标签。
冒泡流是大多数浏览器都支持的。

HTML DOM Event 对象

	(1).节点事件   "p onclick="show()"    /p"
	
	(2).Html DownOn事件。
	
	var pp=document.getElementById('pp');

    (3).HTML W3C标准事件
    pp.addEventListener("click",clickHandler,false)
    function clickHandler(){alert("hello")}
     
     为pp绑定事件监听器，事件为“click”，函数名为“clickHandler”，类型为冒泡流（false）；true为捕获流。
    冒泡流是所有浏览器都支持的 。
