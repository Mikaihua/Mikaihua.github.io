jQuery DOM操作

Dom是Document Object Model的缩写，意思是文档对象模型。DOM是一种与浏览器、平台、语言无关的接口，使用该接口可以轻松访问页面中所有的标准组件。DOM操作可以分为三个方面即DOM Core(核心)、HTM-DOM和CSS-DOM。

一、HTML DOM

(1)、 attr 和 removeAttr

a.attr() 方法设置或返回被选元素的属性值。

			<style>
			.style{background:blue}
			</style>
				$(document).ready(function(){
		  		$("button").click(function(){
		  			$("#div2").attr("style"))}
		  		});
		  		//为 id 为"div2" 的标签设置样式为.style的样式。
		  		
b.removeAttr() removeAttr() 方法从被选元素中移除属性。
                    $("button").click(function(){
                        $("p").removeAttr("id");
                         });

(2)、元素节点： html();text();value();

				a、   ("#div").html("好啦好啦");
		  		b、	("#div2").text("纳尼");  
		  		c、  ("#name").val("请输入姓名")
		  		
		  		// html()中加标签 将其转换成html内容 
		  		//text()中的标签被解析成字符串全部输出。
		  		//val()对所有input 标签可设置其value值。

(3)、文档处理

a、append()和appendTo

				append() 方法在被选元素的结尾（仍然在内部）插入指定内容。
				appendTo()把内容插入到内容的结尾。
				例如：
				         $("#div2").append("hahah")     //#div2中增加内容。
		  			 $("荔枝").appendTo("#div2");    //将内容添加到#div2中。

b、prepend()和prependTo

				prepend() 方法在被选元素的开头（仍位于内部）插入指定内容。
				例如：
				         $("#div2").prepend("hahah")       //#div2中头部增加内容。
		  			 $("荔枝").prependTo("#div2");    //将内容添加到#div2中头部。

c、after()和insertAfter

append是在当前元素内部最后边插入元素,after是在当前元素后面外插入元素.insertAfter() 方法在被选元素之后插入 HTML 标记或已有的元素。

			$("#div2").after("hahahhhah");
		  	$("nihoa").insertAfter("#div");

d、before()和insertBefore()

              before() 方法在被选元素前插入指定的内容。
		insertBefore() 方法在被选元素之前插入 HTML 标记或已有的元素。

e、wrap()和unwarp() 

		wrap()方法把每个被选元素放置在指定的 HTML 内容或元素中。
		unwrap() 方法删除被选元素的父元素。
		例如：$("p").wrap("<div></div>");//把p标签放到div标签里面。
		     $("p").unwrap();//删除p标签的父元素。

f、 empty()和remove()

				$("#div").remove();  //remove 移除 标签
		  		$("#div2").empty();  //empty 移除 标签里面的内容

g、clone()

				 $("body").append($("p").clone());  //复制p标签 并添加到body元素里面去 。

css

a、addClass()

				 此方法向被选元素添加一个或多个类。该方法不会移除已存在的 class 属性，仅仅添加一个或多个 class 属性。
				.style{background:blue}...
				$("div").addClass("style") //向div增加名为style的样式。

b、removeClass()

			removeClass() 方法从被选元素移除一个或多个类。注释：如果没有规定参数，则该方法将从被选元素中删除所有类。
			$("div").removeClass("style")

c、toggleClass()

			toggleClass()用于对样式进行切换
			$("div").removeClass("style")  //点击时对div的样式切换。

d、css()

			css() 方法设置或返回被选元素的一个或多个样式属性。
			$("div").css("background","blue") ;

e、offset() ：offset() 方法返回或设置匹配元素相对于文档的偏移（位置）。
f、position()
g、scrollTop() 方法返回或设置匹配元素的滚动条的垂直位置。
	$(".btn1").click(function(){
         $("div").scrollLeft(100);
                 });

height()，innerHeight(),outerHeight();
height():高度，不包括padding和border
innerHeight() 为匹配的元素集合中获取第一个元素的当前计算高度值,包括padding，但是不包括border。
outerHeight() 包括padding，border和选择性的margin。
